# 🎬 Projeto ScreenMatch 🎬

Este projeto foi desenvolvido para criar um sistema de avaliação e comparação de filmes. Ele permite que os usuários consultem informações de filmes, avaliem e comparem diferentes títulos de acordo com suas preferências.

## 📚 O que este projeto faz?

O sistema fornece uma interface para buscar filmes, registrar avaliações e comparar títulos com base em suas notas. Ideal para quem quer organizar suas listas de filmes.

### 🔍 Como funciona?

1. **🔗 Consulta de Filmes**: O sistema permite que o usuário busque títulos.
2. **📊 Avaliações**: Registra e organiza avaliações feitas pelos usuários.

## 🚀 Funcionalidades

- **🔄 Busca de Títulos**: Permite pesquisar filmes no sistema.
- **⭐ Avaliação de Filmes**: O usuário pode atribuir notas e registrar comentários sobre os filmes.

## 🌐 Consumo da OMDB API

Este projeto utiliza a [OMDb API](http://www.omdbapi.com/) para buscar informações detalhadas sobre os filmes, como título, ano de lançamento, elenco, sinopse e muito mais. A OMDb API oferece uma base de dados rica e completa, permitindo que os usuários façam consultas em tempo real sobre os filmes de seu interesse.

Para utilizar a API, é necessário uma chave de API, que pode ser obtida através de um cadastro gratuito no site.

## 🛠️ Tecnologias Utilizadas

- **Java**: Linguagem usada para a implementação.
- **Maven**: Gerenciador de dependências.
